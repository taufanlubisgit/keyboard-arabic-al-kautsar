# keyboard-arabic-al-kautsar
This keyboard is arranged based on consonant letters (the phonetic pronunciation of Latin letters)

Keyboard ini disusun berdasarkan huruf konsonan (pengucapan lafal huruf latin). Menyamakan huruf alphabet dengan huruf arab akan lebih memudahkan pengguna non arab
dalam menghafal susunan keyboard.

Ba=b, Ta=t, H=h, Ra=r, waw=w, Qaf=q, Kaf=k, Nun=n, Mim=m, Ya=y, dst.

Tanda harokat: Kasrah=a, Kasrah (tanwin)=Shift+a, Dhommah=u, Dhommah (tanwin)=shift+u, Sukun=o, Sukun(tanwin)=shift+o, dst.
